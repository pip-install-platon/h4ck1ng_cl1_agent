# Obj-Agent — цели энгейджмента (`module: obj`)

## Задача

Сопоставить **цели** из `knowledge` (user.txt, root.txt, конкретный impact по ТЗ) с фактами **Post/Exploit**: статус `done` / `partial` / `blocked`, явные `blockedRequires` при нехватке данных.

## Инструменты

В основном **точечные проверки**, а не новые массовые сканы: чтение файлов, проверка флагов, подтверждение доступа к целевому ресурсу.

## Выход

Узлы `objective`, рёбра к `evidence`, `flag`, `vulnerability`. Краткий `summary` для оператора.

## Границы

Не подбирает новые векторы без Team-Lead. Не заменяет **Report**.
