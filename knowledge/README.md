# knowledge/

Статический слой методологии для промпта (до отдельного GraphRAG).

| Что | Роль |
|-----|------|
| **`tactics_stub.md`** | Подмешивается в каждый запрос как блок «Фрагмент методологии» (обрезка по `COPILOT_TACTICS_MAX`). |
| Команда **`knowledge`** в REPL | Показывает тот же фрагмент на экране. |

## Расширение

- Редактируй **`tactics_stub.md`** или добавляй свои секции в том же файле (или замени файл целиком под свой чеклист).
- Альтернативный путь к файлу: **`COPILOT_KNOWLEDGE_PATH`** → абсолютный или относительный путь к markdown.

Дальше — отдельный загрузчик под граф (узлы тактик / CVE / инструментов); текущий формат намеренно плоский.
