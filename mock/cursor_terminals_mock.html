Макет UX — как сейчас в локальном pentest_copilot

Открой этот файл с диска (без git) или подними python mock/serve_mock.py — страница одна, всё статично кроме демо-анимаций. Назначение: сверка «что уже реализовано в терминале (Rich + prompt_toolkit)», а не идеальный дизайн IDE.

Кнопки имитируют события хоста; блоки 2–6 соответствуют текущему REPL.

Соответствие файлам: host.py (подтверждение «Запустить здесь?», колонки результата, превью фона), prompt_menus.py (только F-клавиши для номера bash-блока), runner.py + ui_scrollback.py (стрим / липкая шапка).

1 · Основной промпт чата host.py REPL ›
› что видишь по nmap на 10.10.10.40?

Здесь ввод идёт в обычную строку: цифры не перехватываются — это не меню выбора.

2 · Несколько bash в ответе → номер блока prompt_multi_bash_index

Rich печатает пункты [1]…[n], [0], [a].

[1] первый блок … [2] второй блок … [0] пропустить всё [a] все по очереди (подтверждение перед каждой)
Если установлен prompt-toolkit и не COPILOT_MENU_HOTKEYS=0: F1–F9 — номер блока (до 9), F10 — все, F11 — отмена (как [0]). Блоки 10+ — только вводом цифры в этом же коротком приглашении.
F1–F2 номер блока · F10 все по очереди · F11 отмена · или цифра и Enter › _
3 · Одно подтверждение перед запуском host.py

Отдельного меню [1]/[s]/[3]/… больше нет: команда всегда предлагается к запуску в этом терминале. Режим вывода — переменная окружения COPILOT_FG_MODE=stream|tty|classic (по умолчанию стрим-превью). Фон, окно ОС, stdin-слот — только при trust on и COPILOT_TRUST_*.

Запустить здесь? [Enter/да/y — да · n/0 — пропуск] › _
4 · Во время стрим-превью — живые 2–3 строки COPILOT_STREAM_PREVIEW_LINES
● running foreground · пока идёт процесс, хвост stdout обновляется в «окошке»
Rich Live / chunk — только хвост (полный лог → транскрипт + memory)
…

Режимы: COPILOT_STREAM_MODE=chunk (по умолчанию) или line. Липкая шапка и скроллбэк: COPILOT_STICKY_BANNER, COPILOT_SCROLLBACK_LINES — см. help в REPL.

5 · После exit — две колонки COPILOT_RESULT_COLUMNS

По умолчанию вкл.; выкл. COPILOT_RESULT_COLUMNS=0.

Заголовок/подзаголовок как у Panel(title=…, subtitle=…) в _render_background_slot_previews.

ENV и команды (как в help)
  • COPILOT_FG_MODE — stream | tty | classic для всех запусков «здесь» (если пуст — как COPILOT_TRUST_FG).
  • COPILOT_MENU_HOTKEYS — F-клавиши только при выборе номера bash-блока.
  • COPILOT_RESULT_COLUMNS — две колонки после стрим-команды.
  • COPILOT_BG_PREVIEW, COPILOT_BG_PREVIEW_LINES, COPILOT_BG_PREVIEW_BYTES.
  • stop N / kill N, manual …, task … — без изменений концепта.
