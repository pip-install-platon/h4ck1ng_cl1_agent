Соответствие реальному хосту
  • stop N / kill N — SIGTERM фону; слот пропадает из ● running; панель «процесс остановлен» + транскрипт + разбор логов в memory.
  • manual … — твои действия вне copilot попадают в транскрипт и memory (note), модель их видит.
  • task … — цикл LLM → bash с тем же меню подтверждения; без bash в ответе цикл стоп.
